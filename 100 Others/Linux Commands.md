Check Memory
```
free -m
```
Check Storage
```
ncdu
```
- - -
### Installations
Install from arch-linux
```
sudo pacman -S
```
Install from arch-linux user repository
```
yay -S
```
Update System
```
sudo pacman -Syu
```
Update User Repositories
```
yay -Syu
```
- - -
### Errors
Restart Bluetooth
```
systemctl restart bluetooth 
```
Manually Mount Hard Drive
```
sudo fdisk -l
```
```
sudo mount /dev/sda1 /mnt
```
